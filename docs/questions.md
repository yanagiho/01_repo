# 不明点・質問事項

仕様書（`spec.yaml`）および現状のファイル構成を確認した結果、以下の点について確認または指示が必要です。

1. **チュートリアル動画のパス**:
   - `scenes_flow` に `TUTORIAL_VIDEO` がありますが、具体的な動画ファイルのパスが仕様書やマニフェストに見当たりません。デフォルトで期待されるパス（例：`assets/videos/tutorial.mp4` など）はありますか？

2. **推薦文（RECOMMEND）の選択ロジック**:
   - `RECOMMEND` シーンでの推薦文言は、取得したキャラクタの中からランダムに選ぶ、あるいは優先度の高いものを選ぶなどの具体的なルールはありますか？

3. **PHOTO_TIME の「フレーム」**:
   - 「フレーム(show_frame: true)」とありますが、これは特定の画像ファイル（オーバーレイ）を指しますか？それとも単純な枠線の描画ですか？

4. **ランキングのリセットタイミング**:
   - 「日付変更でリセット」とありますが、アプリ起動時に前日以前のデータを削除する運用でよろしいでしょうか？

5. **アセットのベースパス**:
   - 開発環境（Vite）と納品環境（Electron/Windows 11）の両方で動作させるため、アセットの絶対パス解決ルールを確定させる必要があります。マニフェストに記載された `current` ファイル名を使用して `assets/characters/` などを探索するロジックで実装します。
